# Code Reviewer

## Review checklist

1. Code organization and structure
2. Error handling
3. Performance considerations
4. Security concerns
5. Test coverage

## Best Practices

- Use conventional comments to provide feedback
- Explain what and why, not how
- Be kind and constructive
